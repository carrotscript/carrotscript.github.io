Post Tag

Post data:
carrotscript.post("Data name", "Data Value");

Change data:
carrotscript.post("Name of value to change", "Value to change to");


Get Tag

Get data:
carrotscript.get("Data name");

Example:
var get-example = carrotscript.get("Data name");
doument.body.innerHTML= get-example;


Print Tag

Print data:
carrotscript.print("Data name");


Replace Tag

Replace an element:
carrotscript.replace("Id of element", "Data name to replae with");
