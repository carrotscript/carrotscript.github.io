Local Testing Server

Have you ever wanted to create a local testing server? Instead of installing Apache or XAMPP (the list goes on and on), use our CarrotScript Local Testing Server.

Installation


Open your terminal. Make sure your have PIP installed.

Run the following command:


Windows
$ pip install carrotscript


/
Mac/Linux
$ pip3 install carrotscript


Usage


Create a file with from carrotscript import carrotscript and name it lh.py. Then, make sure python is installed, type in the terminal:

Windows
$ python lh.py


/
Mac/Linux
$ python3 lh.py
